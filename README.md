
# 📚 English জানালা – Full API-Based Vocabulary Web

A dynamic vocabulary web application where users can search, explore, and learn English words with real-time API integration.

---

## 🔗 Live Demo
 https://mou-004.github.io/English-/
 
## 💻 Overview
I recently completed my **English জানালা** project focusing on:

- Real-world API integration  
- Dynamic UI rendering  
- Clean JavaScript logic  

## ⚙️ Features
- 🔹 Dynamic Lesson Buttons loaded from API  
- 🔹 Level-based Word Cards generated dynamically  
- 🔹 Active Button Highlight System  
- 🔹 Real-time UI State Handling (Searching / No Data Found)  
- 🔹 Word Details Modal (API-based dynamic content)  
- 🔹 Dynamic Synonyms Rendering  
- 🔹 Voice Pronunciation using SpeechSynthesis API  
- 🔹 API-based Vocabulary Search (Users can search any English word dynamically)  
- 🔹 English Meaning + Example Fetch from Dictionary API  
- 🔹 English → Bangla Meaning Translation (Translation API integration)  
- 🔹 Enter Key Search Trigger  
- 🔹 Error Handling using try/catch  
- 🔹 Responsive UI using Tailwind CSS + DaisyUI
